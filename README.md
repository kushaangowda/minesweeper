# Minesweeper

Minesweeper game using ReactJS

## Run it locally

1. Clone the repository

```bash
git clone https://github.com/kushaangowda/minesweeper.git
```

2. Install the dependencies

```bash
npm i
```

3. Start the application

```bash
npm start
```
